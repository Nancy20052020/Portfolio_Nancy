# Nancy Verma — Portfolio

Techy, professional portfolio for Nancy Verma (AI & ML · Software Engineering).

## Features

- Light & dark theme toggle
- GSAP scroll animations & floating motion graphics
- Typewriter hero roles
- Project carousel (AutEye, Multi PDF Bot, GEE Remote Sensing P1/P2)
- Experience including Indian Truth Academy (Remote Sensing & GIS) and Grids App LLC
- Responsive glassmorphism sidebar navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
